# 🌐 Подключение домена GoDaddy к GitHub Pages

## ПОШАГОВАЯ ИНСТРУКЦИЯ

---

## ЧАСТЬ 1: ВКЛЮЧЕНИЕ GITHUB PAGES (5 минут)

### Шаг 1.1: Откройте настройки репозитория
1. Откройте браузер
2. Вставьте в адресную строку:
   ```
   https://github.com/aloshkin692/avelle-website
   ```
3. Нажмите Enter
4. Вы попадете на страницу вашего репозитория

### Шаг 1.2: Перейдите в Settings
1. Вверху страницы найдите вкладки: Code, Issues, Pull requests, Actions...
2. Самая правая вкладка - **Settings** (шестеренка ⚙️)
3. **НАЖМИТЕ** на **Settings**

### Шаг 1.3: Откройте Pages
1. Слева появится меню
2. Прокрутите вниз
3. Найдите раздел **Pages** (почти внизу)
4. **НАЖМИТЕ** на **Pages**

### Шаг 1.4: Настройте Source
1. Найдите раздел **Build and deployment**
2. Под **Source** нажмите на выпадающий список
3. Выберите: **Deploy from a branch**
4. Под **Branch** увидите два выпадающих списка:
   - Первый список: выберите **main**
   - Второй список: выберите **/ (root)**
5. **НАЖМИТЕ** синюю кнопку **Save** справа

### Шаг 1.5: Подождите
1. Страница обновится
2. Вверху появится синяя плашка: "GitHub Pages source saved"
3. Подождите 2-3 минуты
4. **ОБНОВИТЕ** страницу (F5 или Ctrl+R)

### Шаг 1.6: Проверьте сайт
1. Вверху появится зеленая плашка с текстом:
   ```
   ✅ Your site is live at https://aloshkin692.github.io/avelle-website/
   ```
2. **СКОПИРУЙТЕ** эту ссылку
3. **ОТКРОЙТЕ** в новой вкладке
4. Ваш сайт должен работать! 🎉

**Если сайт не открывается - подождите еще 5 минут и обновите**

## ЧАСТЬ 2: НАСТРОЙКА DNS В GODADDY (10 минут)

### Шаг 2.1: Войдите в GoDaddy
1. Откройте: https://www.godaddy.com
2. Нажмите **Sign In** (вверху справа)
3. Введите логин и пароль
4. Войдите в аккаунт

### Шаг 2.2: Откройте список доменов
1. Вверху справа нажмите на ваше имя/аватар
2. В меню выберите **My Products**
3. Или перейдите сразу: https://account.godaddy.com/products

### Шаг 2.3: Откройте управление DNS
1. Найдите домен `avelleofficial.com` в списке
2. Справа от домена есть кнопка **DNS** или три точки (⋯)
3. **НАЖМИТЕ** на **DNS** (или ⋯ → **Manage DNS**)
4. Откроется страница DNS Management

### Шаг 2.4: Удалите старые записи
1. Прокрутите вниз до раздела **DNS Records**
2. Вы увидите таблицу с записями
3. Найдите ВСЕ записи с Type = **A**:
   - Наведите на запись
   - Справа появится иконка корзины 🗑️ или карандаш ✏️
   - **НАЖМИТЕ** на корзину
   - Подтвердите удаление
4. Удалите **ВСЕ A записи**
5. Найдите запись **CNAME** с Name = **www**:
   - **УДАЛИТЕ** её тоже
6. Если есть запись **Parked** - тоже удалите

### Шаг 2.5: Добавьте первую A запись
1. Найдите кнопку **Add** или **Add Record** (обычно синяя)
2. **НАЖМИТЕ** на **Add**
3. Откроется форма:
   - **Type**: выберите **A**
   - **Name**: впишите **@** (собачка)
   - **Value** или **Points to**: впишите **185.199.108.153**
   - **TTL**: оставьте **1 hour** или **Default**
4. **НАЖМИТЕ** **Save** или **Add Record**

### Шаг 2.6: Добавьте вторую A запись
1. Снова **Add** → **Add Record**
2. Заполните:
   - **Type**: **A**
   - **Name**: **@**
   - **Value**: **185.199.109.153**
   - **TTL**: **1 hour**
3. **НАЖМИТЕ** **Save**

### Шаг 2.7: Добавьте третью A запись
1. **Add** → **Add Record**
2. Заполните:
   - **Type**: **A**
   - **Name**: **@**
   - **Value**: **185.199.110.153**
   - **TTL**: **1 hour**
3. **НАЖМИТЕ** **Save**

### Шаг 2.8: Добавьте четвертую A запись
1. **Add** → **Add Record**
2. Заполните:
   - **Type**: **A**
   - **Name**: **@**
   - **Value**: **185.199.111.153**
   - **TTL**: **1 hour**
3. **НАЖМИТЕ** **Save**

### Шаг 2.9: Добавьте CNAME запись
1. **Add** → **Add Record**
2. Заполните:
   - **Type**: **CNAME**
   - **Name**: **www**
   - **Value**: **aloshkin692.github.io**
   - **TTL**: **1 hour**
3. **ВАЖНО**: В GoDaddy НЕ ставьте точку в конце!
4. **НАЖМИТЕ** **Save**

### Шаг 2.10: Проверьте результат

Прокрутите вниз и убедитесь, что у вас есть:

```
Type    Name    Value                   TTL
──────────────────────────────────────────────
A       @       185.199.108.153         1 Hour
A       @       185.199.109.153         1 Hour
A       @       185.199.110.153         1 Hour
A       @       185.199.111.153         1 Hour
CNAME   www     aloshkin692.github.io   1 Hour
```

✅ **DNS настроен!**

## ЧАСТЬ 3: ПОДКЛЮЧЕНИЕ ДОМЕНА В GITHUB (2 минуты)

### Шаг 3.1: Откройте GitHub Pages
1. Откройте в браузере:
   ```
   https://github.com/aloshkin692/avelle-website/settings/pages
   ```
2. Или: Репозиторий → Settings → Pages (слева)

### Шаг 3.2: Добавьте Custom Domain
1. Найдите раздел **Custom domain**
2. В поле введите: `avelleofficial.com`
3. **НАЖМИТЕ** кнопку **Save** справа
4. Появится сообщение "DNS check in progress..."
5. Подождите 1-2 минуты
6. Обновите страницу (F5)

### Шаг 3.3: Проверка DNS
1. Если DNS настроен правильно, появится:
   ```
   ✅ DNS check successful
   ```
2. Если ошибка - подождите 10-15 минут и обновите снова
3. DNS может обновляться до 24 часов (обычно 1-2 часа)

### Шаг 3.4: Включите HTTPS
1. Подождите 10-20 минут после успешной проверки DNS
2. Ниже появится чекбокс:
   ```
   ☑️ Enforce HTTPS
   ```
3. **ПОСТАВЬТЕ ГАЛОЧКУ**
4. Ваш сайт получит SSL сертификат (замочек 🔒)

✅ **Домен подключен к GitHub!**

## 🆘 ЧТО ДЕЛАТЬ, ЕСЛИ НЕ РАБОТАЕТ

### Домен не открывается через 4 часа:

**Проверьте DNS:**
1. Откройте: https://dnschecker.org/
2. Введите: `avelleofficial.com`
3. Нажмите **Search**
4. Должны быть зеленые галочки в большинстве мест

**Проверьте записи в GoDaddy:**
1. Вернитесь в GoDaddy → DNS Management
2. Убедитесь, что есть 4 A записи и 1 CNAME
3. Значения должны совпадать с инструкцией

**Проверьте GitHub:**
1. Settings → Pages → Custom domain
2. Должно быть: `avelleofficial.com`
3. Статус должен быть: ✅ DNS check successful

### HTTPS не включается:

1. Подождите 24 часа
2. Галочка "Enforce HTTPS" появится автоматически
3. Если не появилась:
   - Удалите домен (Remove)
   - Подождите 5 минут
   - Добавьте заново
   - Save
